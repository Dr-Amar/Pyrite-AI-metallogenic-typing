# Data folder

The full standardized Excel dataset is not included in the public-safe repository package.

Place the standardized file here for private reproduction:

```text
data/processed/Pyrite_Standarized_data_file_New_Paper.xlsx
```

Expected columns:

```text
Co, Ni, Cu, Zn, Se, Ag, Sb, Pb, Bi, As, Deposit type
```
