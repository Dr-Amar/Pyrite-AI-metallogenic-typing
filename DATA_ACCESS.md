# Data access

The associated article states that data are available on request. For this reason, the public-safe repository template does not include the full standardized Excel dataset.

To reproduce the notebooks privately, place the standardized file here:

```text
data/processed/Pyrite_Standarized_data_file_New_Paper.xlsx
```

Before making a public GitHub repository, confirm:

1. co-author approval;
2. publisher/open-access data conditions;
3. permissions for any compiled third-party literature-derived measurements;
4. whether a reduced demonstration dataset should be released instead of the full compilation.

Recommended public option:

- include code, notebooks, figures, and metadata;
- include a small synthetic or anonymized demonstration file;
- provide contact instructions for full data requests.
